# 💼 Portfólio Simple

Portfólio fictício desenvolvido para apresentar projetos conceituais, conhecimentos técnicos e uma identidade profissional de desenvolvedor front-end.

O projeto foi criado inicialmente durante meus estudos e posteriormente refatorado, recebendo melhorias de estrutura, semântica, acessibilidade, responsividade, identidade visual, interatividade e organização do código.

Todos os dados profissionais, experiências, clientes, estatísticas e projetos apresentados nesta interface são fictícios e possuem finalidade exclusivamente demonstrativa.

## 📱 Responsividade

O projeto foi desenvolvido para funcionar em diferentes tamanhos de tela:

- Desktop;
- Notebook;
- Tablet;
- Smartphone.

A galeria de projetos utiliza:

- Três colunas em telas maiores;
- Duas colunas em tablets;
- Uma coluna em smartphones.

## ♿ Acessibilidade

O projeto recebeu melhorias para proporcionar uma navegação mais inclusiva:

- Link para pular diretamente ao conteúdo principal;
- Textos alternativos nas imagens;
- Atributos ARIA nos controles interativos;
- Estados de foco para navegação por teclado;
- Contraste adequado entre textos e fundos;
- Botões construídos com elementos semânticos;
- Suporte à preferência `prefers-reduced-motion`;
- Links externos com abertura segura em nova aba.

## 🎨 Tema claro e escuro

O portfólio possui alternância entre tema claro e escuro.

O botão de tema atualiza a aparência da página e altera automaticamente o ícone apresentado ao usuário.
